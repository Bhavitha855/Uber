# 🚖 Uber Clone (HTML & CSS)

A responsive **Uber Clone** built using **HTML and CSS**, designed to replicate the user interface of the Uber ride-booking platform.  
This project focuses on modern UI design, layout structuring, and responsive web development concepts.



## 📌 Project Overview

This project recreates the visual design of the Uber website, including layout structure, navigation, and ride-booking interface elements.  
It was developed to improve front-end development skills and understand real-world website design practices using pure HTML and CSS.


## ✨ Features

✅ Uber-inspired modern UI design  
✅ Fully responsive layout  
✅ Navigation bar and landing page  
✅ Clean and structured HTML markup  
✅ CSS Flexbox & Grid layouts  
✅ Mobile-friendly design
